# FpsShooter

`FpsShooter` is an independent Godot 4.6.2 first-person shooter tech probe for the studio.

Current goal:

- prove a simple 3D FPS loop in the editor;
- build a local 1x1 arena against a basic bot;
- keep a traditional FPS rifle baseline while adding special projectile and arena-objective layers gradually;
- use only a light Draxos visual theme, without inheriting gameplay systems from other Draxos projects.

## Current Slice

- PC Windows editor-first.
- No Web/mobile/export target yet.
- Single local arena scene generated by `tools/bootstrap_scene_generator.gd`.
- Player: mouse look, WASD movement, jump, hitscan rifle, RMB Plasma Bolt, overcharge consumption and combat-readable knockback output.
- Arena: `Duel Pit V1`, runtime Health Shard and Overcharge pickups with respawn timers.
- Bot duelista V1+: line-of-sight gated shots, short tell, deterministic aim error, strafe/reposition, vertical awareness, pickup awareness and nearby plasma dodge.
- HUD/feedback: health, bot health, crosshair states, hit/miss, plasma cooldown, overcharge, pickup timers, damage overlay, round state, synthetic audio and runtime primitive shot/impact/knockback/pickup effects.
