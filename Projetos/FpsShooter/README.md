# FpsShooter

`FpsShooter` is an independent Godot 4.6.2 first-person shooter tech probe for the studio.

Current goal:

- prove a simple 3D FPS loop in the editor;
- build a local 1x1 arena against a basic bot;
- start as a traditional FPS before adding special projectile types, jump pads, suspended platforms, void/fall rules and stronger knockback play;
- use only a light Draxos visual theme, without inheriting gameplay systems from other Draxos projects.

## Current Slice

- PC Windows editor-first.
- No Web/mobile/export target yet.
- Single local arena scene generated by `tools/bootstrap_scene_generator.gd`.
- Player: mouse look, WASD movement, jump, hitscan shot and knockback output.
- Bot V1: walks toward the player and fires on cooldown.
- HUD: health, bot health, round state and simple controls.
