# Track 00 - Project Bootstrap

- Status: `ACTIVE`
- Last updated: `2026-06-09`

## Goal

Create the official `FpsShooter` Godot project with local docs, editor-first runtime, validation and portfolio registration.

## Acceptance

- `Projetos/FpsShooter/AGENTS.md` exists.
- `implementation/current-status.md` exists.
- `project.godot` exists and points to the generated arena scene.
- Input actions are bootstrapped by autoload.
- Arena scene is generated by tool, not hand-edited as raw `.tscn`.
- Headless validation runs GUT tests.
- Portfolio docs include `FpsShooter`.

## Out Of Scope

- export/publication;
- Web/mobile;
- multiplayer;
- advanced bot;
- jump pads;
- suspended platforms;
- void/fall arena;
- broad weapon/projectile roster.
