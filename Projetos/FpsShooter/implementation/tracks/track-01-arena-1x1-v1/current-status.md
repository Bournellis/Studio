# Track 01 - Arena 1x1 V1

- Status: `READY_NEXT`
- Last updated: `2026-06-09`

## Goal

Make the first editor-playable 1x1 FPS arena loop against a basic bot.

## Entry Readiness

Track 00 is complete. The next implementation pass can start from the existing editor-playable bootstrap with movement, mouse look, shooting, bot damage, knockback, HUD, restart and pause sensitivity menu already present.

## Planned Acceptance

- player movement and mouse look feel usable;
- player shot damages bot;
- bot walks and shoots;
- basic knockback is visible enough to evaluate;
- HUD communicates health and round state;
- `R` restarts the round;
- manual editor smoke is documented.

## Deferred

- good duel bot behavior;
- special projectiles;
- jump pads;
- suspended platforms;
- void/fall rules;
- export/build packaging.
