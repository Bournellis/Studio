# FPS Playground Publication Readiness

- Last updated: `2026-06-10`
- Status: `FUTURE_REFERENCE`

## Purpose

`FPS Playground` is still PC Windows editor-first. This document records what must exist before treating the project as public or publishable. Track 05 does not perform export or publication.

## Current Position

Ready for:

- local editor playtest;
- technical iteration;
- mode prototyping;
- internal architecture hardening.

Not ready for:

- public demo;
- Windows packaged release;
- Steam page/build;
- multiplayer promise;
- Web/mobile export.

## Readiness Checklist

Before a public build:

- stable project identity and title screen;
- authored or approved placeholder art/audio policy;
- options menu with input/sensitivity/audio controls;
- explicit credits/license doc;
- full Windows export profile;
- smoke-tested packaged executable;
- crash-free 10-15 minute manual playtest;
- performance check on target PC class;
- no editor-only assumptions in runtime path;
- no hidden Draxos canon/progression/backend promises;
- publication-specific validation profile.

## Product Questions To Decide Later

- Is `FPS Playground` a dev lab, a public demo or a real product?
- Does Futebol remain a mode in this project or become a separate prototype?
- Does the Arena Shooter branch become competitive, roguelite, sandbox or pure testbed?
- Will the project ever need online/multiplayer?
- Will it remain runtime-primitives-first or adopt authored art scenes?

## Blocked Scope

The following are not part of Track 05:

- export;
- installer/package;
- store assets;
- final art pass;
- account/save/backend;
- online matchmaking;
- analytics/telemetry;
- release signing.
