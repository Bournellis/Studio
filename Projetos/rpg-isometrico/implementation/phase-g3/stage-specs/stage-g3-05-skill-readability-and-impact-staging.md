# Stage G3-05 - Skill Readability And Impact Staging

## Goal

Make the four baseline skills easier to distinguish at a glance during live combat.

## Required Outcome

- each baseline skill surfaces a distinct on-field read when it is used
- projectile, buff, burst, and leap actions no longer blend together as generic pulses
- the new readability layer stays lightweight and safe for the first slice baseline
- automated validation protects the arena wiring and the new skill feedback layer

## Scope

- runtime 3D skill feedback for the four canonical baseline skills
- arena integration for the new skill staging layer
- tests for the skill readability runtime surface
- smoke updates for manual judgment of skill clarity

## Non-Goals

- new skills, races, or weapons
- damage model redesign
- progression expansion
- online features
- mobile controls
