# Stage G3-01 - Combat Feedback And Telemetry

## Goal

Raise the first arena loop from "works" to "easy to judge and iterate on".

## Required Outcome

- player actions read more clearly in the arena
- the bot exposes a readable melee telegraph before impact
- the HUD helps playtesting by surfacing combat state and recent events
- the result overlay summarizes the round beyond victory/defeat alone
- automated validation protects the telemetry and feedback contracts

## Scope

- combat event telemetry
- bot telegraph readability
- action pulses and response feedback
- richer HUD feedback
- richer result summary

## Non-Goals

- new skills or races
- progression expansion
- online features
- mobile controls
