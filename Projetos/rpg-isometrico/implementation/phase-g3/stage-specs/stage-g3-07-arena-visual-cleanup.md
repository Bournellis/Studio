# Stage G3-07 - Arena Visual Cleanup

## Goal

Reduce floor-layer clutter so the arena stays readable during live combat.

## Required Outcome

- the arena no longer stacks too many active floor guides at once
- manual skill previews compete less with baseline rings and threat surfaces
- buff and burst staging read lighter without losing usefulness
- automated validation keeps the cleanup baseline protected

## Scope

- contextual visibility rules for arena guidance layers
- lighter buff and burst staging
- smoke updates for visual clutter judgment

## Non-Goals

- new mechanics
- content expansion
- balance tuning
- mobile-specific UX
