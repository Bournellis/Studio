# Historical Validation Phase G3 - Combat Productionization

Phase G3 is preserved as the third closed phase in the Godot validation cycle.

## Goal

Turn the arena combat loop into a clearer, more judgeable, and more production-ready runtime surface.

## Focus

- combat feedback and combat readability
- bot telegraph clarity
- round telemetry and result clarity
- HUD feedback that helps playtesting
- automated validation that protects the new combat surface

## Non-Goals

- new content breadth
- online runtime
- mobile support
- long-form progression
