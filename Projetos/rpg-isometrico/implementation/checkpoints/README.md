# Checkpoints

This directory holds balanced evaluation gates between larger implementation phases.

Use checkpoints to decide whether the current playable baseline is ready to expand,
or whether it needs another tightening pass before the next phase opens.

## Current Gates

- `checkpoint-g1-balanced-evaluation.md` approved the first playable baseline for stabilization work.
- `checkpoint-g3-first-slice-viability-acceptance.md` accepts the current Godot slice as a proven migration baseline.
- `checkpoint-g4-local-multi-mode-base-acceptance.md` accepts the Arena/Survival/Boss local base as a credible shared multi-mode foundation.
