# Phase G2 Implementation Map

This file summarizes the intended stable surface for the stabilization phase.

## Target Surface

- stable frontend layout across normal desktop window sizes
- stable arena camera and readable combat framing
- clear player/bot readability and stronger action feedback
- clearer scene ownership for the main playable flow
- validation that protects the first playable baseline from regression
- the same bounded gameplay scope as G1, but with better quality and confidence
