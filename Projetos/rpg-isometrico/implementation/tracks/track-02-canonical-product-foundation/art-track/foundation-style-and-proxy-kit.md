# Foundation Style And Proxy Kit

- Last Updated: `2026-04-21`
- Status: `ACTIVE`

## Purpose

Define the proxy-first art package that supports the canonical bootstrap and tutorial phases without waiting for final production art.

## Phase-Coupled Deliverables

### F01 Meta Foundation And Tutorial

- proxy Blacksmith based on the hammer baseline silhouette
- proxy troll set reused from the existing PvE baseline
- one tutorial-floor environment proxy
- one tutorial copy panel style for onboarding messaging
- one first-skill readability cue for the hammer throw unlock beat

## Pipeline Rules

- author or refine in Tripo3D or equivalent
- clean in Blender
- import to Godot through `glTF 2.0`, preferably `.glb`
- stay inside the budgets defined in `../../../docs/platform-art-and-export-guidance.md`

## Review Checklist

- silhouette reads clearly at the current isometric distance
- proxy materials do not hide combat readability
- import format, scale, pivot, and skeleton rules match the Godot guidance
- the proxy kit is good enough to validate the phase before final art exists
