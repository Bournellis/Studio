## Premise

The game follows a Draxos expedition invading an elemental planet rich in astral energy.

The Draxos operate from a base that resembles a ship, but is plasmated from ether rather than built as conventional technology.

## Hub

The hub is the ship/base interior.

Expected interactive elements:

- captain or commander
- expedition members
- class/training systems
- mission map
- deck/run preparation systems
- archives or narrative systems

The screen should read as one full ship location, not as a list of abstract menus.

## Campaign Goal

The Draxos want to:

- control the planet's astral-energy location
- enslave or dominate elemental beings
- extract a powerful volcano crystal

## Relationship To RPG Turnos

This project may use the same Draxos operation, characters, classes, and final objective.

It does not inherit RPG exploration, campaign progression, combat economy, or board complexity from `rpg-turnos`.
