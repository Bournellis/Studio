# Product Brief

- Last Updated: `2026-05-07`
- Status: `bootstrap direction`

## Identity

`draxos-roguelike-cardgame` is a menu-first roguelike cardgame about a Draxos expedition invading an elemental planet.

The game shares lore, characters, classes, narrative direction, and campaign objective territory with the Draxos material in `rpg-turnos`, but it is a new game with different mechanics.

## Core Promise

The player is a Draxos commander operating from a Draxos ship / ether-plasm base. The hub is a full-screen scene where subordinate NPCs, objects, ship systems, and the communication channel with the Grande Mestre act as buttons and menus.

From the hub, the player opens a mission map, chooses a path through encounters, improves the deck/run state, and pushes toward the final invasion objective.

## What This Is Not

- Not a variant of `rpg-turnos`.
- Not a free RPG exploration game.
- Not a Slay the Spire combat presentation.
- Not a complex RPG Turnos board with routes, elevation, terrain, or neutral slots.

## Pillars

- Class-first pre-run identity: the player is always Draxos, while gameplay changes through class.
- Draxos ship hub as the main menu and narrative anchor.
- Mission map with linear progression and optional steps.
- Small encounters around 1 minute and major encounters around 4-5 minutes.
- Bosses that summon multiple creatures and create escalating board pressure.
- Simple board combat with one player facing one opponent, similar in table presence to MTG.
- Stable 5-card hand with draw-on-play, mana ramp between encounters, and escalating board/deck scale.
- Multiple encounter goals: clear board, duel, waves, protect position, survive turns, boss summoner.

## First Checkpoint

The first checkpoint is official project bootstrap plus a placeholder playable flow: Boot, ShipHub, RunMap, Battle, first clear-board encounter, and first boss-summoner encounter.

It is not expected to be a finished roguelike slice.
