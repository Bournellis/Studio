# Definitions

Source definitions for authored RPG and cardgame content live here.

Current active sources:

- `slice_catalog.json`: Track 01 linear slice with Arcano, Invocador, Necromante, mockup class decks without cost 3 starters, 10 encounter nodes, automatic rewards, front-lane modes, iniciativa, and boss placeholders.
- `visual_assets.json`: Visual support V1 manifest with surface backgrounds, card art paths, card frame paths, and run-map marker positions. PNGs are optional in V1 and use runtime fallbacks when missing.
