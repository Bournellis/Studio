# Track 00 - Project Bootstrap

- Status: `COMPLETE`
- Last updated: `2026-06-09`

## Goal

Create the official `FpsShooter` Godot project with local docs, editor-first runtime, validation and portfolio registration.

## Acceptance

- `Projetos/FpsShooter/AGENTS.md` exists.
- `implementation/current-status.md` exists.
- `project.godot` exists and points to the generated arena scene.
- Input actions are bootstrapped by autoload.
- Arena scene is generated by tool, not hand-edited as raw `.tscn`.
- Headless validation runs GUT tests.
- Portfolio docs include `FpsShooter`.
- Player movement, jump, mouse look, hitscan shot, bot damage, knockback, HUD and restart are present as bootstrap baseline.
- `Esc` pause menu includes a sensitivity slider for editor playtest tuning.
- Known bootstrap editor warnings reported during first tests are fixed.

## Closeout

Track 00 is accepted as the first playable editor baseline. Future work should continue in `Track 01 - Arena 1x1 V1` and focus on making the duel readable and satisfying rather than expanding platform/export scope.

## Out Of Scope

- export/publication;
- Web/mobile;
- multiplayer;
- advanced bot;
- jump pads;
- suspended platforms;
- void/fall arena;
- broad weapon/projectile roster.
