# JogoDaCopa Reuse Map

Allowed reuse:

- Runtime primitive helpers.
- The reused movement/input controller until a dedicated TPS controller is justified.
- Procedural avatar system.
- Feedback/audio primitive patterns.
- Local validation/doc patterns.

Do not import FPS weapons, arena pickups or shooter bot rules from `FpsPlayground` unless a future minigame explicitly adopts them.
