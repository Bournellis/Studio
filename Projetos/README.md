# Projetos

This directory contains active and emerging Godot projects for the studio.

## Project Registry

- `rpg-isometrico/`: campaign-first isometric action RPG. This is the established active implementation surface governed by the current shared canon.
- `rpg-turnos/`: provisional turn-based RPG-cardgame. It may share studio lore, but it owns separate mechanics and starts from a clean visual-agnostic Godot skeleton.

## Agent Rule

Before working in a project, read that project's `AGENTS.md` and `implementation/current-status.md`.

Do not import mechanics from one project into another unless the target project's local docs explicitly adopt them.
