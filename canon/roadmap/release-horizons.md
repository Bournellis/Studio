# Release Horizons

## 1. Purpose of This Document

This document defines the future release horizons of the game as a product.

It exists to separate:

- long-term product-release strategy
- engine-local execution work
- the full eventual project vision

Use this document for shipped-product questions, not for active implementation status.

---

## 2. Rule for Using This Document

Use this document when the question is:

- what the first shipped version should look like
- what should come after the first release
- how the product should expand after launch

Do not use this document to answer:

- what is being implemented right now
- which track or task is active
- what the next local implementation handoff is

Active implementation status lives in `D:\Estudio\Projetos\rpg-isometrico\implementation\current-status.md`.

---

## 3. Release 1 Horizon

Release 1 is intentionally smaller than the full project vision.

Target shape:

- campaign PvE-first
- solo-first, with co-op optional only if it does not compromise the solo baseline
- no public PvP at launch
- a small launch-ready content set
- a narrow but strong kit surface
- clarity and stability over breadth

Likely characteristics:

- 1 race at launch
- 2–3 weapons at launch
- authored campaign as the commercial spine
- Survival, Boss, and Arena Bot as complementary replay or mastery modes where ready

This smaller launch horizon is a shipping strategy, not a change in product identity.

---

## 4. Next Horizon

After Release 1 is stable, the next horizon is:

- deeper PvE variety
- co-op if it did not ship in Release 1
- stronger mastery and replay support
- Free campaign replay/buildcraft support

The game should expand outward from a stable PvE-first base rather than trying to ship every major mode at once.

---

## 5. Later Horizon

After the product is stable enough to support it, the later horizon is:

- private 1v1 duel by direct invite
- broader loadout expression
- more races and weapons
- larger content updates

This is where the released product starts moving more visibly toward the full project identity.

Private duel remains casual/social unless a later product decision explicitly funds stronger competitive online support.

---

## 6. Long-Term Horizon

The long-term horizon is the full project vision:

- multiple races
- broader weapon catalogs
- richer campaigns
- more PvE and PvP modes
- continuing content expansion over time

The exact route may evolve as the product is learned through development and player feedback.

---

## 7. Relationship To Product Vision

`../product/product-vision.md` defines the complete intended identity of the game.

This document defines how the released product is expected to approach that identity over time:

- smaller first release
- broader cooperative and PvE support next
- private duel and wider content growth later
- broader PvP or MOBA only if product traction justifies it

If release planning changes, update this document explicitly.
